Anthropometric Data Analysis
This project analyzes male and female body measurements using NHANES data. It calculates BMI, waist-to-height ratio, and waist-to-hip ratio, standardizes variables using z-scores, and uses plots and correlations to compare body fat distribution between genders.

Tools Used
Python
Pandas
NumPy
Matplotlib

Dataset
NHANES Adult Male and Female Body Measurements (2020)

Purpose
To study obesity-related indicators and understand differences in body composition between males and females.

 Analysis Performed
Data cleaning and preprocessing
Standardisation using z-scores
Scatterplot matrix (pairplot)
Pearson and Spearman correlation analysis
Box-and-whisker plots for ratio comparison
Identification of individuals with lowest and highest BMI
